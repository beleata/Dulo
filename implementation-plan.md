
# План за имплементация на War Thunder Дуло уебсайт

## Общ преглед
Този план описва стъпките за създаване на военен футуристичен уебсайт за ескадрила "Дуло" от War Thunder. Планът е разделен на фази, които следват логическа последователност за ефективна разработка.

## Фаза 1: Подготовка и настройка

### 1.1. Създаване на проектна структура
```
/
├── index.html                 # Начална страница
├── za-nas.html               # За нас
├── istoriya.html             # История на скуадрона
├── polezni-saveti.html       # Полезни съвети
├── prisuedini-se.html        # Присъедини се към нас
├── vruzki.html               # Връзки
├── assets/                   # Ресурси
│   ├── css/
│   │   ├── style.css         # Основни стилове
│   │   ├── hud-nav.css       # HUD навигация стилове
│   │   └── responsive.css    # Responsive стилове
│   ├── js/
│   │   ├── main.js           # Основен JavaScript
│   │   └── animations.js     # Анимации и интерактивност
│   ├── images/
│   │   ├── logo.png          # Лого на Дуло
│   │   ├── hero-bg.jpg       # Фон за начална страница
│   │   └── icons/            # Икони
│   └── fonts/                # Шрифтове
└── texts/                    # Текстово съдържание (вече съществува)
```

### 1.2. Подготовка на ресурси
- Изтегляне на необходимите шрифтове (Orbitron, Bebas Neue, Roboto)
- Подготовка на изображения (лого, фонове, икони)
- Създаване на символ ıYı (ІYІ) като SVG
- Подготовка на камуфлажни текстури

## Технологичен стек

### HTML
- HTML5 семантична структура

- Open Graph тагове за социални мрежи

### CSS
- CSS3 с модерни функции
- Flexbox и Grid за layout
- CSS анимации и преходи

### JavaScript
- Vanilla JavaScript (без фреймуърк)
- ES6+ синтаксис
- Асинхронни операции за форми

- Изображения: Оптимизирани JPEG/WebP за снимки, SVG за икони
