# Структура на War Thunder Дуло уебсайт

## Обща архитектура
```
/
├── index.html                 # Начална страница
├── za-nas.html               # За нас
├── istoriya.html             # История на скуадрона
├── polezni-saveti.html       # Полезни съвети
├── prisuedini-se.html        # Присъедини се към нас
├── vruzki.html               # Връзки
├── assets/                   # Ресурси
│   ├── css/
│   │   ├── style.css         # Основни стилове
│   │   ├── hud-nav.css       # HUD навигация стилове
│   │   └── classic-nav.css   # Класическа навигация стилове
│   ├── js/
│   │   ├── main.js           # Основен JavaScript
│   │   └── animations.js     # Анимации и интерактивност
│   ├── images/
│   │   ├── logo.png          # Лого на Дуло
│   │   ├── hero-bg.jpg       # Фон за начална страница
│   │   ├── tank-bg.jpg       # Фон за танкове секция
│   │   ├── plane-bg.jpg      # Фон за самолети секция
│   │   ├── ship-bg.jpg       # Фон за кораби секция
│   │   └── icons/            # Икони
│   │       ├── tank.svg      # Икона за танкове
│   │       ├── plane.svg     # Икона за самолети
│   │       ├── ship.svg      # Икона за кораби
│   │       └── dulo-symbol.svg # Символ ıYı
│   └── fonts/                # Шрифтове
│       ├── orbitron.woff2    # Основен футуристичен шрифт
│       ├── bebas-neue.woff2  # Шрифт за заглавия
│       └── roboto.woff2      # Шрифт за текст
└── texts/                    # Текстово съдържание (вече съществува)
    ├── hero.txt
    ├── za_nas.txt
    ├── dinastia_dulo.txt
    ├── polezni_saveti.txt
    ├── prisuedini_se.txt
    ├── vruzki.txt
    └── footer.txt
```

## Responsive дизайн

### Мобилна версия (< 768px)
- HUD навигацията се трансформира в "бургер" меню
- Секциите се подреждат вертикално
- Шрифтовете се оразмеряват за по-добра четимост
- Бутоните стават по-големи за лесно докосване

### Таблет версия (768px - 1024px)
- HUD навигацията остава, но бутоните може да са по-малки
- Секциите се адаптират за по-малки екрани
- Някои елементи може да се пренаредят

## Интерактивни елементи

### Анимации
- Сканиращи линии в HUD навигацията
- Пулсиращи индикатори за активна страница
- Hover ефекти с неоново осветяване
- Паралакс ефекти при скролиране

### JavaScript функционалност
- Интерактивно "бургер" меню за мобилни устройства
- Динамично зареждане на съдържание
- Анимации при скролиране
- Валидация на контактна форма
- Превключване между HUD и класическа навигация (опционално)
