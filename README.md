# War Thunder Дуло - Българска общност

## Описание
Военен футуристичен уебсайт за ескадрила "Дуло" от War Thunder, фокусиран върху българската общност. Сайтът съчетава военна естетика с футуристични HUD елементи и камуфлажни мотиви, подходящи за танкови, самолетни и морски битки.

## Структура на проекта

```
/
├── index.html                 # Начална страница
├── za-nas.html               # За нас
├── istoriya.html             # История на скуадрона
├── polezni-saveti.html       # Полезни съвети
├── prisuedini-se.html        # Присъедини се към нас
├── vruzki.html               # Връзки
├── assets/                   # Ресурси
│   ├── css/
│   │   ├── style.css         # Основни стилове
│   │   └── responsive.css    # Responsive стилове
│   ├── js/
│   │   ├── main.js           # Основен JavaScript
│   │   └── animations.js     # Анимации и интерактивност
│   ├── images/               # Изображения (предстои)
│   └── fonts/                # Шрифтове (предстои)
└── texts/                    # Текстово съдържание
    ├── hero.txt
    ├── za_nas.txt
    ├── dinastia_dulo.txt
    ├── polezni_saveti.txt
    ├── prisuedini_se.txt
    ├── vruzki.txt
    └── footer.txt
```

## Основни характеристики

### Дизайн
- **Военен футуристичен стил**: Комбинация от военни елементи с футуристични HUD интерфейси
- **Камуфлажни мотиви**: Смес от зелено, кафяво и сиво, подходяща за всички видове битки
- **Неонови акценти**: Ярки цветове за интерактивни елементи и важна информация
- **Символ ıYı (ІYІ)**: Историческият символ на династия Дуло като централен бранд елемент

### Навигация
- **HUD стил навигация**: Основен препоръчителен вариант с футуристичен изглед
- **Сканиращи линии**: Анимирани линии, които преминават през навигацията
- **Пулсиращи индикатори**: За активна страница и онлайн статус
- **Responsive "бургер" меню**: За мобилни устройства

### Технологии
- **HTML5**: Семантична структура
- **CSS3**: Модерни функции, Flexbox, Grid, анимации
- **JavaScript**: Vanilla JS с ES6+ синтаксис
- **Responsive дизайн**: Оптимизиран за всички устройства

## Как да стартирате проекта

1. Уверете се, че имате инсталиран Python 3.x
2. Отворете терминал и навигирайте до директорията на проекта
3. Стартирайте локален сървър с командата:
   ```bash
   python3 -m http.server 8000
   ```
4. Отворете уеб браузър и отидете на: `http://localhost:8000`

## Браузър съвместимост

- Chrome/Chromium (препоръчителен)
- Firefox
- Safari
- Edge

## Функционалности

### Страници
1. **Начална страница**: Hero секция, предимства на ескадрилата, галерия
2. **За нас**: Информация за ескадрилата и нейните постижения
3. **История**: История на династия Дуло и нейната връзка с ескадрилата
4. **Полезни съвети**: Съвети и стратегии за War Thunder
5. **Присъедини се**: Информация за присъединяване към ескадрилата
6. **Връзки**: Контакти и полезни връзки

### Интерактивни елементи
- **Анимирани числа**: Броячи за статистика
- **Hover ефекти**: Неоново осветяване при посочване
- **Паралакс скролиране**: Различни слоеве се движат с различна скорост
- **Tab функционалност**: За категоризация на съвети
- **FAQ акордеон**: Разгъваеми/сгъваеми въпроси и отговори
- **Форма за контакт**: Валидация и изпращане на съобщения
