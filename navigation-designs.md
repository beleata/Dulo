# Два варианта за навигация на War Thunder Дуло уебсайт

## Вариант 1: HUD стил навигация

### Описание
Навигация в стил Head-Up Display (HUD), вдъхновена от бойните системи в War Thunder. Изглежда като интерфейс на бойна машина с неонови акценти и технологични елементи.

### Визуално представяне
```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ [IYI] ДУЛО WAR THUNDER                                      [STATUS: ONLINE] ● │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  [ЗА НАС]  [ИСТОРИЯ]  [СЪВЕТИ]  [ПРИСЪЕДИНИ СЕ]  [ВРЪЗКИ]                      │
│     ●        ○          ○          ○              ○                           │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### Детайли на дизайна
- **Лого**: Символът "IYI" (ІYІ) от династия Дуло вляво
- **Статус индикатор**: Пулсиращ индикатор "ONLINE" вдясно
- **Навигационни бутони**: С неонови контури и вътрешно запълване при hover
- **Активна страница**: Индикатор ● под активния бутон
- **Неактивни страници**: Индикатор ○ под неактивните бутони
- **Анимации**: Сканиращи линии, които преминават през навигацията
- **Цветове**: Тъмен фон с неоново зелени акценти

### CSS стилове (пример)
```css
.hud-nav {
  background: linear-gradient(135deg, #1a1a1a 0%, #2c3e50 100%);
  border: 2px solid #00ff88;
  box-shadow: 0 0 20px rgba(0, 255, 136, 0.5);
  position: relative;
  overflow: hidden;
}

.hud-nav::before {
  content: "";
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 2px;
  background: #00ff88;
  animation: scan 3s linear infinite;
}

.nav-item {
  color: #00ff88;
  text-shadow: 0 0 10px rgba(0, 255, 136, 0.8);
  transition: all 0.3s ease;
}

.nav-item:hover {
  background: rgba(0, 255, 136, 0.1);
  transform: translateY(-2px);
}

.nav-indicator {
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background: #00ff88;
  box-shadow: 0 0 10px #00ff88;
}

.nav-indicator.active {
  animation: pulse 1.5s infinite;
}
```

## Вариант 2: Класическа хоризонтална навигация

### Описание
По-традиционна хоризонтална навигация, но с военен стил и камуфлажни елементи. Комбинира класическа функционалност с футуристични акценти.

### Визуално представяне
```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  [IYI] ДУЛО - WAR THUNDER БЪЛГАРИЯ                                              │
│                                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────────────┐ │
│  │ ЗА НАС │ ИСТОРИЯ │ СЪВЕТИ │ ПРИСЪЕДИНИ СЕ │ ВРЪЗКИ                        │ │
│  └─────────────────────────────────────────────────────────────────────────────┘ │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### Детайли на дизайна
- **Лого**: Символът "IYI" (ІYІ) с името на ескадрилата
- **Навигационна лента**: С камуфлажен фон и ясни бутони
- **Активна страница**: Подчертан бутон с различен цвят
- **Hover ефекти**: Леко осветяване и промяна на цвета
- **Бордери**: Военни стил бордери около навигационната лента
- **Цветове**: Камуфлажни цветове с оранжеви акценти

### CSS стилове (пример)
```css
.classic-nav {
  background: linear-gradient(90deg, #2d4a2b 0%, #3d2f1f 25%, #2c3e50 50%, #2d4a2b 75%, #3d2f1f 100%);
  border: 2px solid #ff6b35;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.3);
}

.nav-item {
  color: #ffffff;
  border-right: 1px solid rgba(255, 255, 255, 0.2);
  transition: all 0.3s ease;
  position: relative;
}

.nav-item:hover {
  background: rgba(255, 107, 53, 0.2);
  color: #ff6b35;
}

.nav-item.active {
  background: rgba(255, 107, 53, 0.3);
  border-bottom: 3px solid #ff6b35;
}

.nav-item::after {
  content: "";
  position: absolute;
  bottom: 0;
  left: 0;
  width: 0;
  height: 2px;
  background: #ff6b35;
  transition: width 0.3s ease;
}

.nav-item:hover::after {
  width: 100%;
}
```

1. По-добре отговаря на футуристичната военна тематика
